Descripción del Proyecto

Petro Agrónomos es un sistema de gestión integral diseñado para optimizar la planificación y seguimiento de proyectos agronómicos. La plataforma permite la asignación de recursos, gestión de entregas de productos, y visualización de datos geoespaciales para un control preciso de las operaciones en campo.

Esta página es una demostración visual del sistema. Dado que el proyecto está desarrollado con PHP y MySQL, no puede ejecutarse directamente en GitHub Pages. En su lugar, aquí encontrará videos y capturas de pantalla que ilustran sus funcionalidades clave.

Funcionalidades Principales

  • Dashboard Interactivo: Visualización rápida de estadísticas clave como proyectos completados, productos pendientes y entregas realizadas.
  • Gestión de Proyectos: Creación, asignación y seguimiento de proyectos agronómicos con detalles específicos por provincia y bloque.
  • Asignación de Productos: Carrito de compras para seleccionar y asignar productos (fertilizantes, semillas, etc.) a los proyectos.
  • Control de Entregas: Registro y actualización del estado de las entregas de productos a los proyectos asignados.
  • Mapa de Proyectos: Interfaz geoespacial para visualizar la ubicación de todos los proyectos en un mapa interactivo.
  • Reportes y Exportación: Generación de consolidados y reportes en formato Excel para análisis de datos.
  • Gestión de Usuarios: Administración de roles and permisos para diferentes usuarios del sistema.
